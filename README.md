## :star2: Installation

<br>

###### Steps

1. Clone the Repository

```bash
git clone https://github.com/Endy02/terraform_ensup.git
```

2. Go in the app directory

```shell
cd app
```

3. Start terraform plan

```shell
terraform plan -out=tfplanoutput
```

4. Launch the plan

```shell
terraform apply "tfplanoutput"
```
